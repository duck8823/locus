# Spec: #202 draft selection accumulation

## 目的

diff viewer 上での選択を `PromptDraft` に蓄積する。Line / Range / Hunk / File の 4 粒度に対応し、note を付けて貯め、別ペインで一覧・削除できる。

## Rust 側

### AppState

```rust
struct DiffAppState {
    snapshot: PullRequestSnapshot,
    draft: RefCell<PromptDraft>,
    current_anchor: RefCell<Option<SelectionAnchor>>,
    pending_range: RefCell<Option<(u32, Side)>>, // range モード用の始点
}
```

Slint UI は Rc でこの state を共有する。

### state → UI 更新

- `current_anchor_label: SharedString` を更新する `refresh_current_anchor(ui, state)` ヘルパを用意
- `draft_entries: [DraftEntryView]` を rebuild する `refresh_draft_panel(ui, state)` ヘルパ

### selection の解決

- `on_select_line(file_index, line_kind, old_no, new_no)` → side は LineKind::Added/Context ならAfter、Removed なら Before
- `on_select_hunk(file_index, hunk_index)` → Granularity::Hunk
- `on_select_file()` → selected file に対して Granularity::File
- `on_extend_to_range()` → current が Line のときだけ、次のクリックで Range になる `pending_range` フラグを立てる

### draft 操作

- `on_add_to_draft(note: SharedString)` → current_anchor と note から DraftEntry を作り push、refresh_draft_panel
- `on_remove_draft_entry(index)` → draft.remove(index)、refresh_draft_panel
- `on_clear_current_selection()` → current_anchor = None、refresh_current_anchor

## Slint 側

### DiffLineView を拡張

```slint
export struct DiffLineView {
    kind: int,                    // 0=Context 1=Added 2=Removed -1=HunkHeader
    is-hunk-header: bool,
    hunk-index: int,
    old-line-no: string,
    new-line-no: string,
    content: string,
}
```

### DraftEntryView

```slint
export struct DraftEntryView {
    label: string,    // "a.rs:10 (After)" のような短い表示
    note: string,     // 省略可
}
```

### DiffViewerWindow を拡張

- プロパティ追加:
  - `in property <string> current-anchor-label`
  - `in property <[DraftEntryView]> draft-entries`
- コールバック追加:
  - `callback select-line(int, int, string, string)` (file_index, line_kind, old_no, new_no)
  - `callback select-hunk(int, int)` (file_index, hunk_index)
  - `callback select-whole-file()`
  - `callback extend-to-range()`
  - `callback add-to-draft(string)` (note)
  - `callback remove-draft-entry(int)` (index)
  - `callback clear-current-selection()`

### UI レイアウト（既存の 2 ペイン構成を拡張）

```
┌──────────────────────────────────────────────────────────────┐
│ Header                                                        │
├──────┬──────────────────────────────────────┬───────────────┤
│ File │ Diff content (clickable lines)        │ Draft panel   │
│ list │                                       │ 280px         │
│      │                                       │               │
│      │                                       │ entries list  │
│      │                                       │ each with [×] │
│      │                                       │               │
│      ├──────────────────────────────────────┤               │
│      │ Selection controls (bottom strip)    │               │
│      │   anchor label | [File][Range]       │               │
│      │   note input   | [Add] [Clear]       │               │
└──────┴──────────────────────────────────────┴───────────────┘
```

各 diff line に TouchArea を被せ、click で select_line を呼ぶ。hunk header 行は `is-hunk-header=true` で別 TouchArea に select_hunk を割り当てる。
