# Spec: #205 parser-adapter + Semantic IR contract

## 目的

ADR 0004 の parser-adapter + 共通 Semantic Change IR 境界を Rust 側の型として置く。具体実装は v0.2 以降（tree-sitter-go など）で積むため、v0.1 時点では契約とフォールバック stub のみ。

## 追加モジュール src/semantic/

- adapter.rs : ParserAdapter trait, ParsedSnapshot, ParserDiffItem, ParserDiffResult
- ir.rs      : SemanticChange, SemanticChangeGroup, SymbolKind, ChangeType, UnsupportedFileAnalysis
- fallback.rs: FallbackLineParserAdapter (1 file = 1 file-level symbol の stub)
- mod.rs     : re-export

## テスト

- ParserAdapter trait が dyn として成立すること (object safety)
- FallbackLineParserAdapter の parse / diff が stub 動作すること
- IR 型の基本的な equality / default
