# Dogfooding Run Artifacts

> 日本語: [README.ja.md](README.ja.md)

This directory stores JSON artifacts generated by:

```bash
npm run dogfood:run
```

## Filename format

- `run-<ISO timestamp with ':' '.' replaced by '-'>.json`
- Example: `run-2026-03-12T08-12-45-123Z.json`

## JSON shape

```json
{
  "generatedAt": "2026-03-12T08:12:45.123Z",
  "commands": [
    {
      "command": "npm run demo:data:reseed",
      "stdout": "...",
      "stderr": ""
    }
  ],
  "metrics": {
    "generatedAt": "2026-03-12T08:12:45.120Z",
    "jobsFilePath": ".../jobs.json",
    "global": {
      "totalJobs": 0,
      "terminalJobs": 0,
      "averageDurationMs": null,
      "failureRatePercent": null,
      "recoverySuccessRatePercent": null
    },
    "byReview": []
  }
}
```

## Usage notes

- Keep raw artifacts for trend tracking and incident investigation.
- If the artifact includes sensitive command output, sanitize before sharing externally.
